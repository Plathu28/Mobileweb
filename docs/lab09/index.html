ตัวอย่างหน้าจอการใช้งาน (Android)

1. หน้าเริ่ม (Start Page)

หน้าเมนูหลักพร้อมปุ่ม Start และการตั้งค่าเสียงแนะนำ

2. ขณะกำลังนับ (Counting)

แอปตรวจจับการยกแขนและนับจำนวนรอบแบบเรียลไทม์

3. หน้าสรุปสถิติ (Statistics)

แสดงจำนวนรวมและเวลาที่ใช้เมื่อจบการออกกำลังกาย
